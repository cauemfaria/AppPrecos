# AppPrecos - Frontend (PWA)

Este é o frontend do aplicativo AppPrecos, construído com React, Vite e Tailwind CSS. Ele é um Progressive Web App (PWA) que pode ser instalado em dispositivos móveis.

## 🚀 Início Rápido

1.  **Instalar dependências:**
    ```bash
    npm install
    ```

2.  **Configurar Variáveis de Ambiente:**
    Crie um arquivo `.env` nesta pasta:
    ```env
    VITE_API_BASE_URL=https://seu-backend.onrender.com/api
    ```

3.  **Desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Build de Produção:**
    ```bash
    npm run build
    ```

## 🏗️ Estrutura

-   `src/pages`: Páginas principais (Scanner, Mercados, Lista, Ajustes).
-   `src/components`: Componentes reutilizáveis e Layout.
-   `src/services`: Cliente de API (Axios).
-   `src/store`: Gerenciamento de estado global (Zustand).
